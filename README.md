# 基于同源模型差值的功能插件提取与软叠加注入

本仓库包含论文的 PDF 文件及其实验代码。论文提出了一种通过计算同源模型（MiniCPM-1B Base / SFT / RL）间的参数差值来提取“功能插件”的方法，并采用软叠加（缩放系数 α）注入基座，在诗歌生成任务上验证了有效性。

## 📄 论文摘要

大模型在预训练、指令微调（SFT）和强化学习（RL）等阶段中积累了不同的能力增量，但这些增量通常无法分离复用。本文通过计算同源模型间的参数差值提取“功能插件”，并采用软叠加注入基座。在 MiniCPM-1B 的诗歌生成任务上，我们发现：  
① 直接提取 RL-Base 差值优于链式减法；  
② MLP 层是安全注入载体，Attention 层易引发复读崩溃；  
③ 最佳方案为 α=0.3 的 MLP-only 注入。

## 🔬 核心实验结果

- **最佳配置**：MLP-only 插件 + 缩放系数 α=0.3  
- **成功输出示例**（五言绝句）：
  ```
  冬雪漫天飞，
  雪花飘飘落，
  寒风凛凛冽，
  雪花漫天飞。
  ```
- **反例**：硬叠加（α=1.0）导致无限循环，链式减法导致语义崩塌。

## 🛠️ 快速复现

1. 下载 MiniCPM-1B 的 Base / SFT / RL 模型权重。
2. 运行 `extract_delta.py` 提取 RL-Base 差值。
3. 运行 `inject_plugin.py` 注入插件（可调整 α 值）。
4. 运行 `test_injection.py` 生成诗歌，观察输出质量。
